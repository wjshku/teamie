# Teamie - 一个人就是一支队伍

## 产品简介

Teamie 是一款专为科研创业者设计的高效项目管理工具，帮助研究生、研究员在一个界面中快速了解各项目的整体进展，并通过周进展分析报告实现高效决策与复盘。

## 核心功能

### 📥 Notion 项目导入
- 支持导入 Notion HTML 导出文件（含子页面）
- 自动解析页面层级、会议记录、任务、文档等信息
- 无需手动录入，快速接入现有项目资料

### 📊 周进展分析报告
基于六个维度生成结构化进展报告：
- **动机与方向** - 为什么这样做
- **已完成事项** - 本周实际完成了什么
- **未完成事项** - 计划要做但没做的事情
- **内部反思** - 没做的原因分析
- **外部反馈** - 成效和用户反馈
- **下一步计划** - 下周的具体行动计划

### 📱 统一项目视图
- 所有项目一览，实时显示完成率
- 支持移动端和桌面端使用
- 快速切换项目查看详细进展

## 目标用户

- 科研工作者（研究生、研究员）
- 创业者
- 需要同时管理多个复杂项目的人员

## 产品价值

- **降低认知负担**：在一个界面掌握所有项目的全景
- **精准决策**：减少决策前的信息收集时间
- **高效复盘**：结构化的进展分析帮助快速总结经验
