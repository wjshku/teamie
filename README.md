# Teamie - 一个人就是一支队伍

## 产品简介

Teamie 是一款专为科研创业者设计的高效项目管理工具，帮助研究生、研究员在一个界面中快速了解各项目的整体进展，并通过周进展分析报告实现高效决策与复盘。

## 核心功能

### Notion 项目导入
- 支持导入 Notion HTML 导出文件（含子页面）
- 自动解析页面层级、会议记录、任务、文档等信息
- 无需手动录入，快速接入现有项目资料

### 周进展分析报告
基于六个维度生成结构化进展报告：
- **动机与方向** - 为什么这样做
- **已完成事项** - 本周实际完成了什么
- **未完成事项** - 计划要做但没做的事情
- **内部反思** - 没做的原因分析
- **外部反馈** - 成效和用户反馈
- **下一步计划** - 下周的具体行动计划

### 统一项目视图
- 所有项目一览，实时显示完成率
- 支持移动端和桌面端使用
- 快速切换项目查看详细进展

## 目标用户

- 科研工作者（研究生、研究员）
- 创业者
- 需要同时管理多个复杂项目的人员

## 产品价值

- **降低认知负担**：在一个界面掌握所有项目的全景
- **精准决策**：减少决策前的信息收集时间
- **高效复盘**：结构化的进展分析帮助快速总结经验

## 技术栈

- **后端**：FastAPI + Python
- **前端**：HTML/CSS/JavaScript
- **AI**：OpenAI GPT-4
- **数据存储**：本地 JSON 文件
- **设计工具**：Figma/Mermaid

## 安装和运行

### 环境要求
- Python 3.8+
- OpenAI API Key

### 安装步骤

1. **克隆项目**
   ```bash
   git clone <repository-url>
   cd teamie
   ```

2. **设置环境变量**
   ```bash
   cd backend
   cp .env.example .env
   # 编辑 .env 文件，填入你的 OpenAI API Key
   ```

3. **运行启动脚本**
   ```bash
   cd ..  # 返回项目根目录
   ./start.sh
   ```

   脚本会自动：
   - 创建 Python 虚拟环境
   - 安装依赖包
   - 启动后端服务

4. **访问应用**
   打开浏览器访问：http://localhost:8000

### 使用说明

1. **导入项目**
   - 点击"导入新项目"
   - 选择 Notion 导出的 HTML 文件
   - 系统会自动分析并生成第一周进展报告

2. **查看进展**
   - 在项目列表中选择项目
   - 查看结构化的周进展报告
   - 使用编辑模式修改报告内容

3. **编辑内容**
   - 点击"编辑内容"按钮进入编辑模式
   - 可以添加、删除、修改各项内容
   - 点击"保存修改"提交更改
